# 项目成本智能评估系统 - 演示版

## 🎯 快速开始

### 方法1: 使用启动脚本（推荐）
```bash
./start-demo.sh
```

### 方法2: 手动启动
```bash
cd public
python3 -m http.server 3000
```

## 🌐 访问地址

- **演示页面**: http://localhost:3000/demo.html
- **首页**: http://localhost:3000/index.html

## 🚀 停止服务

在终端按 `Ctrl+C` 停止HTTP服务器。

## 🔧 故障排除

### 端口被占用
```bash
lsof -i :3000
kill -9 <PID>
```

### Python不可用
```bash
# 检查Python版本
python3 --version

# 如果没有Python3，尝试
python --version
```

### 页面无法访问
1. 确认服务器已启动
2. 检查端口是否正确
3. 确认浏览器地址为 http://localhost:3000/demo.html

## 📞 支持

如有问题，请检查：
1. Python3是否已安装
2. 端口3000是否可用
3. 浏览器是否支持HTML5
